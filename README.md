# Web-applications-project
1) As a guest, I want to search for available rooms by specifying my check-in and check-out dates.
2) As a guest, I want to be able to modify or cancel my booking.
3) As an admin, I want to see all bookings and modify or cancel them.
4) As a guest, I want to see basic information about available rooms, such as the room type, price etc.
5) As a hotel manager, I want to receive a message that someone has booked a room.
